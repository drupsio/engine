# Installation

## Requirements

- `Python >= 3.6`
- `pip`
- `venv (recommended)`
- `RabbitMQ Server`
- `Redis Server`

## Production

```shell
$ pip install drups
```

## Development

- Clone the repository: `git clone git@github.com:drupsio/engine.git`
- Go to the project directory: `cd engine`
- Install pip dependencies: `pip install -r requirements/dev.txt`
